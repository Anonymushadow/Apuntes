_para acceder a este pones el nmbre del objeto + . + el nombre del atributo

UpCasting = 
_convertir un objeto de x a z -> z = padre y x = hijo
_Vehiculo a = new auto(); -> convierte el objeto a de auto a Vehiculo
_Vehiculo b = new Vehiculo();
_Vehiculo b = a(); -> a = objeto auto
_generalizar para hacer lo mismo para muchas subclases, todos los vehiculos aceleran pero no todos tienen aire
_a = especifico
_v = general



DownCasting = 
_Vehiculo V = A; -> en la variable V (en la clase vehiculo) se guardo el objeto a (de la clase Auto); --->  upcasting
_Auto A2 = (Auto) V; -> en A2 se guardo V conviertiendolo de vuelta a un objeto de clase Auto;
_asegurarte siempre de convertir el objeto al mismo o bjeto -> V = auto, no hay qe guardarlo como moto xque tira error, no podes convertir un auto en una moto
