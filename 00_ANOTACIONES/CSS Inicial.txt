0_INTRODUCCION
------------------------------------------------------------------------------------------------------
Archivo CSS: en este archivo le indicamos a la pagina como queremos que se vea cada elemento html 
    .por ejemplo si queremos un texto de color rojo, si queremos que la foto sea mas chica, etc 

Estructura:
    .un archivo css se compone de selectores y estilos
    .los electores se utiliza para seleccionar uno o varios elementos HTML de una página web

Como indico donde esta mi archivos con estilos?: en el html podes añadir estilo de 3 maneras
    1_Podes poner estilos css entre las etiquetas <style></style>
    2_Podes poner una etiqueta <link> que indica donde se encuentra este archivo con estilos
    3_Podes poner en la etiqueta html style="" y dentro de esto pones los estilos (estilos en linea)
    .Como indico a que elemento deseo añadirle tales estilos?: esto se realiza mediante los Selectores
        .Selectores: son nombres en css que se utilizan para seleccionar uno o varios elementos HTML en una página web
        .Tipos de selectores:
            1_Etiqueta: podemos indicar que etiquetas editar 
                por ejemplo: 
                    p {} --> aca definimos como se veran todos los elemntos p
            2_Clase: podemos indicar como se veran todos los elemntos que tengan tal Clase
                Por ejemplo: 
                    .contenedores {} --> aca definimoscomo se veran todos los elementos con la clase contenedores
                    para indicar que estamos seleccionando segun la clase ponemos un . antes de la clase 
            3_Id: podemos indicar como se vera el elemento que tenga tal Id
                Por ejemplo: 
                    #imagePerfil {} --> aca definimoscomo se vera el elemento con el id imagePerfil 
                    para indicar que seleccionamos segun el id ponemos un # antes del id
        .Extra: a veces varios elementos deben compartir estilos pero no comparten la clase ni el id ni la etiqueta,
        para estos casos podemos crear los estilos y llamar a cada elemnto que tendra ese estilo
            por ejemplo:
                
                h1,
                h2,
                p {
                    background-color: red;
                    color: blue;
                }

            en este caso tanto los h1, como los h2 y los p tendran un fondo rojo con letra azul
Especificidad: se trata de que tipo de selector, orden y estilo tiene mas importancia sobre otros
    el orden y el tipo de estilos influyen en el resultado
    Orden:
        .CSS se ejecuta de arriba para abajo
        .la linea 1 se ejecuta antes que la 2 y asi
            .por ejemplo si en la linea 40 pongo que el texto tendra color rojo pero en la linea 41 le digo que
            el texto tendra color azul primero se va a pintar el texto de rojo y despues se pintara de azul, 
            provocando que nosotros lo veamos de color azul
    Gerarquia de Especificidad (de menor a mayor):
        1_pseudo-elementos y elementos
        2_clases, Pseudoclases y atributos
        3_ID
        4_estilos en linea
        5_ !important: podemos ponerle al estilo !important y eso significa que sin importar el tipo de selector ni la posicion, este sera el qeu se vera si o si    

Metodología BEM: Se trata de a cada clase y id ponerle nombres descriptivos separador por dos guiones bajos
    por ejemplo 
        contenedor__imagen
        header__info__redes
        header__info__redes__iconos

1_UNIDADES
------------------------------------------------------------------------------------------------------
Unidades Relativas: dependen de la medida de otra cosa
    em: Relativo al tamaño de fuente del elemento. 
        1 em = tamaño del mismo estilo pero en el contenedor 
        por defecto 1 em suele ser 16px 
        por ejemplo: si un contenedor tiene 5px de tamaño de letra y a su contenido le decimos que su tamaño
        de letra sera 2 em significa que sera igual a 10px (2 em seria en este caso 2 * 5px)
    rem: Relativo al tamaño de fuente del elemento raíz (generalmente el tamaño de fuente del elemento <html>).
    %: Porcentaje relativo al tamaño del elemento padre.

Unidades Absolutas:
    px: Píxeles, una unidad absoluta de tamaño.
    cm: Centímetros.
    mm: Milímetros.
    in: Pulgadas.
    pt: Puntos (1/72 de pulgada).
    pc: Picas (12 puntos).

Unidades de Vista:
    vw: Porcentaje del ancho de la ventana gráfica.
    vh: Porcentaje de la altura de la ventana gráfica.
    vmin: El porcentaje del ancho o alto de la ventana, dependiendo de cuál sea el menor.
    vmax: El porcentaje del ancho o alto de la ventana, dependiendo de cuál sea el mayor.

porque usamos medidas relativas?: al adaptar la pagina a tamaños mas chicos o mas grandes, la letra debe adaptarse
    por ejemplo si hacemos una pagina web para pc va a tener letra muy grande y si adaptamos la pagina a moviles
    la letra se vera muy grande, para eso usamos las medidas relativas, estas se achican o agrandas respecto a otras cosas

Viewport width vs Viewport height:
    Viewport: es la pantalla entera
    bordes que no se ve la pagina en si
    Viewport width (vw): esto indica cuanto queremos que mida el elemento respecto al ancho del viewport
        100vw: significa que mide exactamente lo mismo que el ancho del viewport
    Viewport height (vh): esto indica cuanto queremos que mida el elemento respecto al alto del viewport
        100vh: significa que mide exactamente lo mismo que el alto del viewport

3_ELIMINAR ESTILOS POR DEFECTO
------------------------------------------------------------------------------------------------------
Normalize:
    .Normalize.css es un archivo CSS que se utiliza para establecer un conjunto consistente de estilos
    iniciales en todos los navegadores
    .Normalize.css no elimina todos los estilos por defecto, sino que establece reglas consistentes 
    para que los estilos se vean igual o muy similares en todos los navegadores.

Padding 0 y Margin 0:
    .padding: 0; y margin: 0; son reglas CSS que establecen el relleno y el margen de todos los elementos 
    en una página en 0. Esto elimina cualquier espacio en blanco por defecto alrededor de los elementos.
    .Es común usar esto como parte de un "reset" o "normalización" de estilos para asegurarse de que no 
    haya espacios no deseados entre elementos.

Box-Sizing: Border-Box:
    .box-sizing: border-box; es una propiedad CSS que controla cómo se calcula el tamaño total de un elemento, 
    incluyendo su contenido, relleno y borde.
    .Cuando se establece en border-box, el ancho y la altura del elemento incluyen el tamaño del relleno y 
    el borde. Esto significa que si defines un elemento con un ancho de 100px y un padding de 10px, el 
    contenido real del elemento tendrá un ancho de 80px (100px - 2 * 10px) en lugar de 100px.

5_PADDING Y MARGIN
------------------------------------------------------------------------------------------------------
Shorthands: un shorthand es una abreviacion de varias propiedades
padding y margin son Shorthands
    padding: se utiliza para definir el espacio entre cada borde del elemento y su contenido
        por ejemplo: 
            padding: 4px;  
            indica que tendra 4px entre su contenido y su borde
            padding: 4px 2px; 
            indica que tendra 4px abajo y arriba pero tendra 2px a la derecha y a la izquierda
        shorthand: padding es la abreviacion de padding-top + padding-right + padding-bottom + padding-left
            tambien podemos poner paddin y una medida por cada lado
                padding: 4px 13px 2px 10px;
    margin: se utiliza para definir el espacio que habra entrer el elemento y cualquier otro elemento por fuera de este
        por ejemplo: 
            margin: 4px; 
            indica que habran 4px al rededor del elemento donde no podra haber otro elemento
            margin 3px 9px; 
            indica que habran 3 pixeles arriba y abajo del elemento y 9 pixeles a la derecha e izquierda del elemento 
        shorthand: margin es un shorthand de margin-top + margin-right + margin-bottom + margin-left
            tambien podemos poner margin y una medida por cada lado
                margin: 4px 3px 2px 1px;

10_POSITION
------------------------------------------------------------------------------------------------------
position: se trata de como posicionar el elemnto, segun su posicion adquiere ciertas propiedades
    el position afecta el flujo del html (orden en el cual añadimos los elementos)
    el flujo de html afecta el como se ven los elementos, el elemento añadido ultimo esta por encima de los primeros
    al posicionarlo obtiene 4 propiedades nuevas:
        top: --> Define la distancia entre el borde superior del elemento y el borde superior del elemento contenedor
        left: --> Define la distancia entre el borde izquierdo del elemento y el borde izquierdo del elemento contenedor
        right: --> Define la distancia entre el borde derecho del elemento y el borde derecho del elemento contenedor
        bottom: --> Define la distancia entre el borde inferior del elemento y el borde inferior del elemento contenedor
static --> es un elemento comun, sin posicion
relative --> el espacio del elemento se mantiene intacto, por ejemplo si le ponemos un top 500px, el elemento se va a mover hacia abajo
pero el espacio donde estaba este va a estar vacio, y el elemento en si va a estar por encima de los otros elementos
absolute --> es como el relative, pero la diferencia es que cuando se posiciona su espacio es ocupado por los demas elementos 
que le rodean
sticky --> se debe definir una posicion, el elemento sera estatico hasta que llegue a la posicion definida, al llegar ahi,
el elemento se quedara fijo, lo que significa que  si la pagina se sigue moviendo el elemento se quedara en ese espacio 
por encima de todo, como un stiker (mayormente usado para un navbar que al bajar se queda fijo en un lugar)
fixed --> funciona como un stiker, se queda encima de todo, no ocupa un espacio y siempre esta en el mimso lugar
(comunmente usado para las publicidades que aparecen abajo en la pantalla)

11_Z-INDEX
------------------------------------------------------------------------------------------------------
¿Que es?: se trata de definir la posicion en el eje Z
con esta propiedad indicas que tan adelante o atras esta un elemento respecto a los demas
por ejemplo el elemento 1 esta debajo del elemento 2 pero si le añadimos un z-index 1 al elemnto 1 ahora estara por encima
solo funciona cuando la caja esta posicionada (osea tiene un pistion)
Consejo: añadir z-index de 100 en 100, de 50 en 50, o a eleccion pero no de 1 en 1
    por ejemplo: 
        si tengo 20 elementos, y le añado un z index 1 al elemento 20, 2 al 19 y asi quedando el ultimo elemento abajo de todo y el
        primero encima, si de repente deseo que el numero 5 este encima del 8 no puedo modificarlo con facilidad porque
        los numeros del 1 al 20 los tengo ocupados, deberia cambiar todos, en cambio si voy de 100 en 100 puedo ponerle un z-index 
        150 y ya estaria entre el 100 y el 200
