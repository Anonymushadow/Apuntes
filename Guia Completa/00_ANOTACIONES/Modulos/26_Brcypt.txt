26_BCRYPT
---------------------------------------------------------------------------------------------
多Que es?: es un modulo de terceros utilizado para encriptar contrase単as
instalar: npm install bcrypt
utilizar: const bcrypt = require('bcrypt');
palabras reservadas:
    bcrypt.hashSync() --> generar un hash seguro a partir de una contrase単a en texto plano
    bcrypt.compareSync() --> verificar si una contrase単a en texto plano coincide con un hash almacenado previamente
    bcrypt.genSaltSync() --> generar una nueva sal aleatoria antes de crear un nuevo hash
